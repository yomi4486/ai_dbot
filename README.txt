LMStudioや、ChatGPTなどのLLMとDiscordBotを使って会話するやるです。
ChatGPT-APIを使用する場合、アクセストークンの記載は適宜行ってください。

.envについて
TOKEN = DIscordBotのトークン
APPLICATION_ID = DiscordBotのクライアントID
base_url = OpenAIライブラリに対応したLLMのAPIのURL
DEEPL_API_KEY = DEEPL翻訳のAPI_KEY

各サービスの利用規約に則ってご使用ください。

使用しているライブラリは、下記コマンドでインストールできます
pip install -r requirements.txt
